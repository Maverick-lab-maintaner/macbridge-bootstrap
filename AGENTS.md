# Repository Instructions

These instructions apply to every task in this repository.

## Documentation Rule

After every major milestone, update the documentation before closing the work.

Required docs pass:

- update `HISTORY.md` with the exact chronological record of what changed, what broke, and how it was fixed
- update or extend `docs/macbridge-story.md` or the nearest learning-oriented doc so the milestone is understandable to a reader
- keep the write-up specific enough that someone can learn the toolchain, failure modes, and fixes from it

This is mandatory for major milestones, not optional cleanup.
